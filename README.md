# ToDoList_RoomDB
The ToDoList is an Android application that use for daily tasks. User can create task, edit/update task and delete task.

<h1>Built With</h1>
<ul>
<li>Java</li>
<li>AsyncTask</li>
<li>RoomDatabase</li>
<li>Material Design</li>
<li>MVC</li>
</ul>
